VARIABLES:
-------------------
->variables are named memory loaction used to store data or values
 and it can change N no of times during execution.

 1.variable declaration
 2.variable initialisation
 3.variable utilisation
 4.variable re- initialisation
 5.variable re- declaration

in js we can define variables in 3 ways
----------------------------------------
1.let
    ->let variables act as a local scope / block scope
    ->variable declaration and initialisation is possible in same / different line
    ->re- initialisation is possible but not re declaration
    ->we cant hoist the let variables

2.var
    ->var variables acts like a global scop(because var present in windows objectively)
    ->variable re-declaration and initialization is same/different line is possible
    ->we can hoist the var variable
    ->variable declaration and initialisation is possible in same / different line

3.const
    ->const variables acts like a global final(because once initialisation we cant modify data/values)
    ->variable re-declaration and initialization is same line is not possible 
    ->we cant hoist the var variable
    ->variable declaration and initialisation is possible in same  line but not in different line




    Hoisting:
    ------------
      ->Utilizing the variables before declaration.
      ->Hoisting is possible only through 'var' variable.
