MongoDB : 
---------
    ->MongoDB is the NoSQL used to store data in form of Document formate


    Database: 
        syntax : use databaseName

Collection : 
------------
    ->Collection is the group of documents in MongoDB 
    ->syntax : 
            db.createCollection('collectionName')

Document : 
----------
    ->Document(JSON/BSON) contains group fields 


    ->Insert Document 
        1. syntax 
                    db.collectionName.insertOne(document)

        2.syntax 
                    db.collectionName.insertMany([document1, document2,...])

Retrieve Documents /data: 
-------------------------
    1. findOne()  ->used to Retrieve one document at a time
            syntax: 
                db.collectionName.findOne({filter condition}, {projection query})


                projections : 
                        ->projections are used to Retrieve required fields 
                        ->to display fields pass value ->1    (_id: 1)
                        ->if don't want to display value ->0   (fields:0)

Update Document Fields : 
-----------------------
    1. updateOne() 
        ->updateOne() is used to Update single document at a time 

            syntax :

    2. updateMany()
        ->updateMany() is used to Update Multiple documents at a time 
            syntax : 
                    db.collectionName.updateMany({filter condition},{updated query})
